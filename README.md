# Jaweria Khan - Portfolio

Modern portfolio website for Jaweria Khan, Front-End Developer. Built with vanilla HTML, CSS, and JavaScript. No frameworks, 100% custom code.

**Live Demo:** https://jaweria-khan.vercel.app

### **Features**
- **Responsive Design**: Mobile-first layout with fluid grids + aspect-ratio cards
- **3D Interactions**: Tilt effect on project cards using vanilla JS 
- **Contact Form**: Connected to Google Sheets via Google Apps Script - no backend needed
- **Dark Theme**: Glassmorphism + gradient accents
- **Animations**: Intersection Observer reveal animations + typewriter effect
- **Optimized Images**: WebP format for fast loading

### **Tech Stack**
| Tech | Use |
| --- | --- |
| HTML5 | Semantic structure |
| CSS3 | Custom properties, Grid, Flexbox, Animations |
| Vanilla JS | DOM manipulation, Form handling, 3D tilt |
| Google Apps Script | Serverless form submission |
| Google Sheets | Contact form database |
| Vercel | Hosting + CI/CD |
